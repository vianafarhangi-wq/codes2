# Contributing to codes2

Thanks for considering contributing! Ideas for contributions that would be
welcome:

- Add additional image transforms (rotate, crop, color adjustments).
- Add batch processing or preserve EXIF metadata.
- Add type checking with mypy and improve docstrings.

To get started:

1. Fork and clone this repository.
2. Create a new branch for your feature/fix.
3. Add tests for your change.
4. Ensure `pytest` passes locally: `pytest -q`.
5. Submit a pull request describing the change and why it's useful.
